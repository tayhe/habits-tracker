# Habits_Tracker 前端规范

## 1. 概述

单文件 HTML 应用，复刻 Notion 风格的学习习惯追踪系统。

### 角色差异

| 功能 | parent | child |
|---|---|---|
| 每日追踪视图 | ✅ 单日切换 | ✅ 七日视图 |
| 每周汇总视图 | ✅ | ✅ |
| 任务管理 | ✅ | ❌ |
| 编辑历史记录 | 任意日期 | 仅最近3天 |

---

## 2. 视觉规范

### 配色（Notion 风格）
- 背景: `#FFFFFF`（主内容区）、`#F7F6F3`（侧边/Header）
- 文字: `#37352F`（主文字）、`#9B9A97`（次要文字）
- 边框: `#E9E9E7`
- 科目标签色:
  - 英语: `#D6E4FF`（蓝底）`#2D7ED8`（蓝字）
  - 数学: `#FFE6D6`（橙底）`#D4621C`（橙字）
  - 语文: `#D6F4E0`（绿底）`#1F8D43`（绿字）
- 进度表情: 🐟🐡/🐟/😭

### 字体
- 主字体: `-apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`
- 标题: `font-weight: 600`, `font-size: 14px`
- 正文: `font-size: 14px`

### 间距
- 基础单位: `8px`
- 卡片圆角: `6px`
- 内边距: `12px ~ 16px`

---

## 3. 页面结构

### Header
- 左侧: Logo（🐟）+ "小鱼干习惯追踪"
- 右侧: 用户名 + 角色标签 + 登出按钮

### 视图切换栏
- 按钮组: [每日追踪] [每周汇总] [任务管理（仅parent）]
- 当前视图高亮（背景色 `#F7F6F3`，加粗）

### 每日追踪视图
**parent 模式:**
- 顶部: `< 2026-05-21 >` 日期切换
- 主体: 按科目分组卡片，每科下任务列表
- 每行: 任务名标签 + 完成状态勾选框 + 收益值
- 底部: 总收益 + 表情

**child 模式:**
- 七日水平滚动视图
- 第1~4列为历史（灰色只读）
- 第5~7列为最近3天（绿色边框可编辑）
- 今日在最右侧，底部显示表情和总收益

### 每周汇总视图
- 顶部: 周次选择器 `2026-W21`
- 主体: 三栏（英语/数学/语文），每栏显示完成天数/总天数 + 进度条 + 总收益
- 底部: 总计

### 任务管理视图（仅 parent）
- 任务列表（可编辑名称/收益/周最低次数/排序权重）
- 新增任务表单

### 登录页
- 居中卡片: Logo + 用户名输入 + 密码输入 + 登录按钮
- 演示账号提示: taihe / cat123（parent）、miaomiao / fish123（child）

---

## 4. 交互规范

- 点击任务行切换完成状态 → 实时 PUT /api/v1/records
- 切换日期/周次 → 重新请求数据
- child 用户在第1~4天点击 → 弹窗提示"历史记录不可修改"
- 表情计算: 三科均100%→🐟🐡；三科均≥50%→🐟；其他→😭

---

## 5. 技术要求

- 单文件 `index.html`，内联 CSS + JS
- 纯原生 JS，不使用框架
- Cookie 自动携带 session_token
- 响应式: 小屏幕任务卡片垂直堆叠
- CORS 已由后端配置 `allow_origins=["*"]`
